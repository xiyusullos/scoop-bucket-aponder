# aponder-bucket

Personal Scoop bucket for archiving specific application versions.

## Usage

```powershell
scoop bucket add aponder C:\Users\x\my-bucket
scoop install aponder/chrome
scoop install aponder/wechat
```

## Manifests

| App | Version | Source |
|-----|---------|--------|
| Chrome | 138.0.7204.184 | [ulixee/chrome-versions](https://github.com/ulixee/chrome-versions) |
| WeChat | 3.9.9.43 | [tom-snow/wechat-windows-versions](https://github.com/tom-snow/wechat-windows-versions) |
